# Newblog of Leunico

> 这是利用Symfony2核心组件以及其他轻量组件开发的博客。

### 技术栈
- Symfony2四大核心组件（http-foundation、routing、http-kernel、event-dispatcher）
- 超轻量级容器-pimple
- Laravel5的两个组件（database、validation）
